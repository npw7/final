What If

Create two token designs exploring a speculative fiction topic.

Technical Notes

Illustrator, Line Tool, Shape Tool, Text Wrap

Project Documentation

> Write up speculative fiction narrative to base design off of
> Concept art
> Draw tokens in Illustrator

Reflection

Thinking about the narrative for this project was very fun to do. I am a fan of speculative fiction, so getting to explore that in a project was a cool way to learn more design skills. The token designs were fun to make, although I need more Illustrator practice in general. I had a lot of ideas that I knew how to execute in other programs, but in Illustrator it was more of a struggle. My second, monochrome token design was my favorite. I find the simple visuals more charming.
