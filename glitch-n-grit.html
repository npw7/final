Glitch N Grit

Create three different pieces that explore the use of compression and pixelation as art forms. Create a website using HTML to display these pieces..

Technical Notes

Clip Studio Paint, Phoenix Code (HTML, CSS, JavaScript).

Project Documentation

> Create the art pieces in my drawing software
> Experiment with strategically pixelating certain parts of the works
> Start learning how to use HTML
> Implement the CSS code
> Briefly touch on JavaScript functions

Reflection

This project was interesting to me, mostly for the HTML section. I've never worked with HTML before, aside from filling in code templates on websites before; but at that point I had no idea why things were happening or what each line meant. This project actually made me realize that I enjoy learning how to code things such as this. I had a lot of fun figuring out how to use CSS to make my page look how I wanted it to. My greatest challenge was trying to make a functional button that took you to the Cow GIFs page. I really wanted to make a pretty button that changed colors when you hovered over it. I had no idea that adding such a button would require figuring out how to use JavaScript functions. The button took me the longest to figure out compared to all parts of the project, but I found it rewarding in the end when I was able to make it work the way that I wanted it to. I think that in the future I will continue to learn how to code for websites on my own time. I want to learn how to make more functional pieces with JavaScript.
